# Project 3 
* This project is a tool to test, update, and create the car shopping API on the browser. <br>
* Car shopping API implements full CRUD operations using SQLite server database. <br>



## Technologies Used 
* Node <br>
* Express <br>
* React <br>
* Lucidcharts ERD <br>
